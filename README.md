# Kanban Board - Task Management App

A simple and interactive Kanban Board built using HTML, CSS, and JavaScript.  
This project allows users to manage tasks efficiently by organizing them into Todo, In Progress, and Done columns.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Drag and drop tasks between columns
- Persistent data using LocalStorage
- Clean and responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage API

## How It Works

1. Users can create tasks in any column.
2. Tasks can be dragged between columns.
3. Tasks can be edited or deleted anytime.
4. All tasks are saved in LocalStorage, so data remains even after refreshing the page.
